# Marshall Islands Data Files
This directory contains 13 mandatory data files for the Republic of the Marshall Islands, including country overview, YAML data files for various sectors, and progress tracking.

## Files Included
1. COUNTRY.md - Country overview and key facts
2. README.md - This file
3. country.yaml - Basic country metadata
4. geography.yaml - Geographic data
5. demographics.yaml - Population and demographic stats
6. economy.yaml - Economic indicators
7. government.yaml - Government structure and leadership
8. culture.yaml - Cultural practices and traditions
9. symbols.yaml - National symbols (flag, anthem, etc.)
10. history.yaml - Historical timeline
11. infrastructure.yaml - Transportation, energy, and utilities
12. education.yaml - Education system data
13. healthcare.yaml - Healthcare infrastructure and stats
