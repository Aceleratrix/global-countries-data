# New Zealand Data Package 🇳🇿
This directory contains structured data for New Zealand as part of the Global Countries Data project.

## Contents
- `COUNTRY.md`: Basic country overview
- `README.md`: This file
- 11 YAML data files covering:
  1. `general.yaml`: Core country identifiers and status
  2. `geography.yaml`: Location, area, borders, maps
  3. `demographics.yaml`: Population, demonyms, languages
  4. `economy.yaml`: GDP, currency, economic indicators
  5. `government.yaml`: Political structure, UN membership
  6. `culture.yaml`: Languages, native names, translations
  7. `infrastructure.yaml`: Capital, timezones, driving side
  8. `symbols.yaml`: Flag, coat of arms, national symbols
  9. `contact.yaml`: Phone codes, postal codes, maps
  10. `sports.yaml`: FIFA code, Olympic participation
  11. `metadata.yaml`: Data source, update timestamp

## Data Source
All data fetched from REST Countries API v3.1 (https://restcountries.com/v3.1/alpha/NZ)
GDP data: $0.06T (139th largest economy)
