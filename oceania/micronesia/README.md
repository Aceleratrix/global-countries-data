# Micronesia (Federated States of Micronesia) Country Data

This directory contains structured data for the Federated States of Micronesia, organized into 11 YAML datasets plus core documentation.

## Files
- `COUNTRY.md`: Basic country overview and key facts
- `README.md`: This file, describing the directory structure
- 11 YAML data files:
  - `geography.yaml`: Physical and political geography data
  - `demographics.yaml`: Population, ethnicity, language stats
  - `government.yaml`: Political structure and governance
  - `economy.yaml`: Economic indicators and sectors
  - `culture.yaml`: Cultural practices, traditions, and heritage
  - `education.yaml`: Education system and literacy stats
  - `healthcare.yaml`: Healthcare infrastructure and outcomes
  - `infrastructure.yaml`: Transportation, energy, and utilities
  - `environment.yaml`: Environmental features and challenges
  - `security.yaml`: Defense and law enforcement
  - `tourism.yaml`: Tourism industry and attractions
