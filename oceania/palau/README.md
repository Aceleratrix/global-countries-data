# Palau Data Directory

This directory contains 13 mandatory structured data files for Palau (🇵🇼), the 173rd largest economy globally with $0.05T GDP.

## Files Included
1. `COUNTRY.md` - Main country overview
2. `geography.yaml` - Physical geography and location data
3. `demographics.yaml` - Population, ethnicity, and language data
4. `economy.yaml` - Economic indicators and GDP data
5. `government.yaml` - Political system and governance data
6. `culture.yaml` - Cultural practices and traditions
7. `infrastructure.yaml` - Transportation and utilities data
8. `education.yaml` - Education system and statistics
9. `healthcare.yaml` - Healthcare system and public health data
10. `environment.yaml` - Environmental protections and biodiversity
11. `tourism.yaml` - Tourism industry data
12. `history.yaml` - Historical timeline and key events
13. `religion.yaml` - Religious demographics and practices

All data is sourced from verified public databases including REST Countries API, World Bank, and UNESCO.
