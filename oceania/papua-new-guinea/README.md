# Papua New Guinea Country Data
This folder contains structured, verified data for Papua New Guinea (Oceania region) as part of the global-countries-data repository.

## Files Included
1. `COUNTRY.md` - Key country facts summary
2. `README.md` - This file
3. 11 YAML data files:
   - `country.yaml` - Basic country identifiers and attributes
   - `geography.yaml` - Terrain, climate, regions, borders
   - `demographics.yaml` - Population, ethnicity, religion, literacy
   - `economy.yaml` - GDP, industries, trade, currency
   - `government.yaml` - Political system, leadership, elections
   - `culture.yaml` - Languages, traditions, holidays, arts
   - `history.yaml` - Pre-colonial, colonial, independence, modern era
   - `infrastructure.yaml` - Transport, energy, telecommunications
   - `education.yaml` - Literacy, enrollment, universities
   - `healthcare.yaml` - Life expectancy, health indicators, services
   - `environment.yaml` - Biodiversity, protected areas, issues

## Data Sources
- World Bank Open Data
- United Nations Statistics Division
- Papua New Guinea National Statistical Office
- ISO 3166 Maintenance Agency
- Central Intelligence Agency (CIA) World Factbook
