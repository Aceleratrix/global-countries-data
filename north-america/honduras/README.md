# Honduras - Country Data
This directory contains structured data for Honduras as part of the Global Countries Data repository.

## Files
- `COUNTRY.md`: Main country overview
- 11 YAML data files covering: basic info, geography, demographics, economy, government, culture, history, transportation, education, healthcare, environment
- `README.md`: This file

## Data Sources
All data is sourced from verified public databases including UN, World Bank, CIA World Factbook, and national Honduran institutions.
