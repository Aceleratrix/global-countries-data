# Costa Rica Data Repository
This directory contains 13 mandatory data files for Costa Rica (94th largest economy, $0.10T GDP), sourced from the REST Countries API and verified official records.

## Files Included
1. COUNTRY.md - Comprehensive country overview
2. README.md - This file
3. basic-info.yaml - Core identity and administrative data
4. geography.yaml - Geographic and border details
5. demographics.yaml - Population and demographic stats
6. economy.yaml - Economic indicators including GDP data
7. currency.yaml - Currency details (Costa Rican colón)
8. language.yaml - Language and translation data
9. government.yaml - Political status and international codes
10. infrastructure.yaml - Transport, postal, and telecom data
11. symbols.yaml - Flags, coat of arms, and national symbols
12. culture.yaml - Demonyms, cultural traits, and social data
13. international.yaml - UN, FIFA, IOC, and other international affiliations

## Data Source
All core data sourced from REST Countries API v3.1: https://restcountries.com/v3.1/name/Costa%20Rica
GDP rank and value per task context.
