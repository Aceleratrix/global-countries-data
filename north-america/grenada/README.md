# Grenada Data Directory

This directory contains 13 mandatory files with comprehensive data about Grenada (🇬🇩), the 165th largest economy globally with $0.05T GDP.

## Files Structure
- `COUNTRY.md` - Country overview and key facts
- `README.md` - This file
- 11 YAML data files:
  - `basics.yaml` - Core country identifiers and codes
  - `geography.yaml` - Geographic and location data
  - `demographics.yaml` - Population and demographic info
  - `economy.yaml` - Economic indicators including GDP
  - `government.yaml` - Government system and structure
  - `language.yaml` - Official and native languages
  - `currency.yaml` - Currency details
  - `timezones.yaml` - Timezone information
  - `borders.yaml` - Bordering countries (none for Grenada)
  - `flags.yaml` - Flag images and descriptions
  - `regional.yaml` - Regional classification data

## Data Source
All data is sourced from the [REST Countries API](https://restcountries.com/) and verified official sources.
