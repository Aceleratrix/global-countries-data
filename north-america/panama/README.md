# Panama Country Data 🇵🇦
This directory contains 13 verified data files for Panama (72nd largest economy, $0.16T GDP).

## Files Included
1. COUNTRY.md - Core country overview
2. economy.yaml - GDP, industries, currency
3. demographics.yaml - Population, ethnic groups
4. geography.yaml - Area, borders, climate
5. government.yaml - Political system, leadership
6. culture.yaml - Language, religion, holidays
7. infrastructure.yaml - Canal, transport, utilities
8. energy.yaml - Energy sources, electricity access
9. education.yaml - Literacy, enrollment data
10. healthcare.yaml - Health metrics, spending
11. tourism.yaml - Visitor numbers, attractions
12. history.yaml - Key historical events
13. README.md (this file) - Directory overview

All data sourced from World Bank, IMF, and Panama INEC (2024-2025).
