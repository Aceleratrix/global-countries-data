# Trinidad and Tobago Country Data

This directory contains structured data files for Trinidad and Tobago (🇹🇹) as part of the Global Countries Data repository.

- `COUNTRY.md`: Overview of basic country information
- `basics.yaml`: Core country identifiers and metrics
- `geography.yaml`: Geographic details and features
- `demographics.yaml`: Population, ethnicity, and language data
- `government.yaml`: Political structure and leadership
- `economy.yaml`: Economic indicators and industries
- `culture.yaml`: Cultural practices, festivals, and arts
- `tourism.yaml`: Tourism attractions and statistics
- `education.yaml`: Education system and metrics
- `healthcare.yaml`: Healthcare system and statistics
- `transportation.yaml`: Transport infrastructure
- `energy.yaml`: Energy production and consumption
- `telecommunications.yaml`: Telecom and internet infrastructure
