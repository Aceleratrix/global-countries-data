# Saint Vincent and the Grenadines Data Directory

This directory contains the 13 mandatory structured data files for Saint Vincent and the Grenadines, as per the global-countries-data repository standard.

## File List
1. `COUNTRY.md` - Country overview and key facts
2. `README.md` - This directory documentation
3. `economy.yaml` - Economic indicators and trade data
4. `geography.yaml` - Physical geography and terrain data
5. `government.yaml` - Political system and governance structure
6. `demographics.yaml` - Population and social demographics
7. `culture.yaml` - Language, traditions, and cultural practices
8. `infrastructure.yaml` - Transportation, energy, and telecom infrastructure
9. `education.yaml` - Education system and literacy rates
10. `healthcare.yaml` - Public health and medical infrastructure
11. `energy.yaml` - Energy production and renewable targets
12. `environment.yaml` - Environmental conservation and challenges
13. `tourism.yaml` - Tourism industry and attractions

All data is sourced from verified public datasets including the REST Countries API, World Bank, and national statistics offices.
