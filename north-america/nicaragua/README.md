# Nicaragua Country Data
This directory contains structured data for Nicaragua, organized into YAML files covering key domains, plus a country overview (COUNTRY.md).

## Files
- COUNTRY.md: High-level country overview
- general.yaml: General country metadata
- geography.yaml: Geographic and physical data
- demographics.yaml: Population and demographic statistics
- economy.yaml: Economic indicators and data
- politics.yaml: Political system and governance information
- culture.yaml: Cultural practices, festivals, and traditions
- history.yaml: Historical timeline and events
- education.yaml: Education system and statistics
- health.yaml: Healthcare system and public health data
- infrastructure.yaml: Transportation, energy, and infrastructure assets
- environment.yaml: Environmental features and conservation data
