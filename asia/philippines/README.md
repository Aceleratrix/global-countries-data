# Philippines Data Repository
This directory contains structured data for the Republic of the Philippines, compliant with the global-countries-data 13-file standard.

## Files
1. COUNTRY.md - High-level country overview
2. README.md - This file
3. basic_info.yaml - Core country metadata
4. geography.yaml - Geographic data
5. demographics.yaml - Population statistics
6. economy.yaml - Economic indicators
7. government.yaml - Political structure
8. culture.yaml - Cultural heritage
9. education.yaml - Education system
10. healthcare.yaml - Healthcare infrastructure
11. infrastructure.yaml - Transport, energy, etc.
12. environment.yaml - Environmental data
13. tourism.yaml - Tourism statistics
