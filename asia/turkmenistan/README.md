# Turkmenistan Country Data
This directory contains 13 mandatory files with verified real data about Turkmenistan.

3. country_info.yaml - Core country metadata

5. demographics.yaml - Population and demographic data
6. economy.yaml - Economic indicators and structure
7. government.yaml - Political system and governance
8. history.yaml - Historical timeline
9. culture.yaml - Cultural traditions and practices
10. infrastructure.yaml - Infrastructure overview
11. education.yaml - Education system data
12. health.yaml - Healthcare and public health data
13. tourism.yaml - Tourism information

All data is sourced from verified sources: UN, World Bank, IMF, and official Turkmen government records.
