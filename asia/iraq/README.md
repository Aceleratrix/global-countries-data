# Iraq Data Directory

This directory contains 13 mandatory data files for Iraq (🇮🇶), the 35th largest economy globally with a GDP of $0.37 trillion.

## Files Included

1. **COUNTRY.md** - Main country overview and key facts
2. **economy.yaml** - Economic indicators, trade, and financial data
3. **demographics.yaml** - Population, ethnic groups, languages, and social data
4. **geography.yaml** - Physical features, borders, climate, and natural resources
5. **government.yaml** - Political system, administrative divisions, and governance
6. **culture.yaml** - Cultural heritage, religion, arts, and traditions
7. **energy.yaml** - Oil, gas, electricity, and renewable energy data
8. **education.yaml** - Education system, literacy, and institutions
9. **healthcare.yaml** - Health indicators, infrastructure, and challenges
10. **infrastructure.yaml** - Transportation, telecom, and utilities
11. **military.yaml** - Armed forces, budget, and security
12. **tourism.yaml** - Tourism industry, attractions, and statistics
13. **README.md** - This file

## Data Sources

All data reflects 2023 estimates from:
- World Bank
- International Monetary Fund (IMF)
- United Nations (UN)
- Iraqi Government Ministries
- OPEC
- CIA World Factbook
