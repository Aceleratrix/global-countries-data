# Oman Data Repository
This folder contains 13 mandatory data files for Oman 🇴🇲, including country overview, 11 sector-specific YAML datasets, and this README.

## Files Included
1. `COUNTRY.md` - Country overview and key facts
2. `README.md` - This file
3. `economy.yaml` - Economic indicators
4. `demographics.yaml` - Population and social data
5. `geography.yaml` - Geographic and climatic data
6. `government.yaml` - Political and governance data
7. `culture.yaml` - Cultural and religious data
8. `education.yaml` - Education sector data
9. `health.yaml` - Healthcare and demographic health data
10. `infrastructure.yaml` - Transport and utility infrastructure
11. `tourism.yaml` - Tourism sector data
12. `trade.yaml` - Trade and export/import data
13. `environment.yaml` - Environmental indicators

All data is sourced from World Bank, UN, and Omani government reports (2023-2024 estimates).
