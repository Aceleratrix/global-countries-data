# Yemen Data Directory

This directory contains 13 mandatory data files for Yemen (🇾🇪), 121st largest economy with $0.07T GDP.

## Files
1. COUNTRY.md - Core country overview
2. README.md - This file
3. country.yaml - Basic country metadata
4. economy.yaml - Economic indicators
5. geography.yaml - Geographic data
6. people.yaml - Demographic information
7. government.yaml - Governance details
8. infrastructure.yaml - Infrastructure data
9. culture.yaml - Cultural information
10. history.yaml - Historical overview
11. military.yaml - Military data
12. environment.yaml - Environmental info
13. symbols.yaml - National symbols

All data sourced from REST Countries API v3.1 and verified public datasets.
