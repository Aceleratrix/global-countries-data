# Timor-Leste (East Timor) Data Directory

This directory contains 13 mandatory files with verified real data about Timor-Leste (East Timor), including country overview, YAML datasets for 11 key domains, and this README.

## Files
- `COUNTRY.md`: Country overview and quick facts
- `README.md`: Directory description (this file)
- `basic_info.yaml`: Core country identification and basic stats
- `geography.yaml`: Physical and locational data
- `demographics.yaml`: Population, ethnicity, religion, and health metrics
- `economy.yaml`: Economic indicators, industries, and trade
- `politics.yaml`: Government structure, leaders, and electoral data
- `culture.yaml`: National symbols, traditions, and cultural practices
- `history.yaml`: Historical timeline from colonization to present
- `infrastructure.yaml`: Transportation, energy, and telecom data
- `education.yaml`: Education system and enrollment statistics
- `healthcare.yaml`: Public health and medical infrastructure data
- `security.yaml`: Military, police, and defense information

## Data Source
All data sourced from World Bank, CIA World Factbook, UN Statistics Division, and official Timor-Leste government reports (2023-2024).
