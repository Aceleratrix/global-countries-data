# East Timor (Timor-Leste) Country Data

This directory contains 13 mandatory files for East Timor, including the main country profile (COUNTRY.md), this README, and 11 structured YAML datasets covering key aspects of the country.

## Files
- COUNTRY.md: Main country overview
- README.md: Directory metadata
- 11 YAML files: Structured data in geography, demographics, economy, government, history, culture, language, religion, flag, anthem, and national symbols
