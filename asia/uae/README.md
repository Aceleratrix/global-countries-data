# UAE Country Data
This directory contains 13 mandatory data files for the United Arab Emirates, structured per global-countries-data standards.
## Files Included
1. COUNTRY.md - Core country overview
2. README.md - This file
3. legal/internet-law.yaml - Internet regulations
4. legal/labor-law.yaml - Labor market laws
5. fiscal/tax-codes.yaml - Taxation framework
6. financial/banks.yaml - Banking sector data
7. telecom/phone-formats.yaml - Telecom numbering plans
8. demographics/population.yaml - Population statistics
9. tech/ai-cloud-services.yaml - AI and cloud infrastructure
10. business/company-sizes.yaml - Business size classifications
11. business/real-estate.yaml - Real estate market data
12. business/import-export.yaml - Trade statistics
13. marketing/digital-marketing.yaml - Digital marketing landscape
