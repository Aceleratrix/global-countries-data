# North Korea Data Repository
This directory contains structured data for the Democratic People's Republic of Korea (North Korea), complying with the global-countries-data repository standards. All files use verified real-world data as of 2024.

## Files
- COUNTRY.md: Main country overview
- demographics.yaml: Population, age structure, literacy, etc.
- economy.yaml: GDP, industries, trade data
- geography.yaml: Area, borders, terrain, climate
- government.yaml: Political structure, leaders, administrative divisions
- history.yaml: Key historical events post-1945
- culture.yaml: Language, traditions, arts
- military.yaml: Armed forces size, budget, equipment
- infrastructure.yaml: Transport, energy, telecommunications
- education.yaml: School system, enrollment rates
- healthcare.yaml: Medical system, life expectancy, disease stats
- environment.yaml: Protected areas, pollution, natural resources
