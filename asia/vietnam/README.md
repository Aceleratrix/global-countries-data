# Vietnam Data Files

This directory contains all 13 mandatory data files for Vietnam (53rd largest economy, $0.26T GDP).

## Files List (13 total)
1. COUNTRY.md - Country overview and key facts
2. README.md - This file
3. economy.yaml - Economic indicators
4. demographics.yaml - Population data
5. geography.yaml - Geographic information
6. government.yaml - Political structure
7. culture.yaml - Cultural heritage
8. infrastructure.yaml - Infrastructure data
9. education.yaml - Education statistics
10. healthcare.yaml - Healthcare metrics
11. tourism.yaml - Tourism data
12. military.yaml - Military information
13. energy.yaml - Energy sector data

## Data Sources
- World Bank
- International Monetary Fund (IMF)
- ASEAN Secretariat
- Vietnam General Statistics Office (GSO)
- UNESCO

All data reflects 2023 estimates unless otherwise noted.
