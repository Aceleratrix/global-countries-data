# Cambodia Data Directory
This directory contains 13 structured data files for Cambodia (🇰🇭), including country overview, 11 domain-specific YAML datasets, and this README.

## Files
- COUNTRY.md: Country overview
- README.md: This file
- economy.yaml: Economic indicators
- demographics.yaml: Population and social data
- geography.yaml: Geographic and border info
- government.yaml: Political system and leadership
- culture.yaml: Language, traditions, cuisine
- infrastructure.yaml: Transport, energy, connectivity
- education.yaml: Literacy, enrollment, universities
- healthcare.yaml: Health indicators and services
- tourism.yaml: Tourism stats and attractions
- trade.yaml: Export/import and trade partners
- environment.yaml: Forests, biodiversity, issues
