# Uzbekistan Data Directory

This directory contains structured data for Uzbekistan, organized into 13 mandatory files as per the global-countries-data standard.

## Files Included
1. COUNTRY.md - Core country profile
2. README.md - This file
3. geography.yaml - Geographic data
4. demographics.yaml - Population and demographic stats
5. economy.yaml - Economic indicators
6. politics.yaml - Political structure
7. culture.yaml - Cultural information
8. history.yaml - Historical overview
9. infrastructure.yaml - Infrastructure details
10. education.yaml - Education system
11. healthcare.yaml - Healthcare data
12. environment.yaml - Environmental information
13. tourism.yaml - Tourism-related data

All data is verified from official sources including the World Bank, UN, and Uzbek government publications.
