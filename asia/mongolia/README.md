# Mongolia 🇲🇳
Data repository for Mongolia, containing 13 mandatory files: COUNTRY.md, README.md, and 11 structured YAML datasets with verified real-world data.

## Files List
- [COUNTRY.md](./COUNTRY.md): Country overview and key facts
- [basic_info.yaml](./basic_info.yaml): Basic country metadata
- [geography.yaml](./geography.yaml): Geographic and terrain data
- [demography.yaml](./demography.yaml): Population and demographic statistics
- [economy.yaml](./economy.yaml): Economic indicators and industry data
- [politics.yaml](./politics.yaml): Political structure and governance
- [culture.yaml](./culture.yaml): Cultural heritage and traditions
- [history.yaml](./history.yaml): Historical timeline and key events
- [education.yaml](./education.yaml): Education system data
- [healthcare.yaml](./healthcare.yaml): Healthcare system metrics
- [transport.yaml](./transport.yaml): Transportation infrastructure
- [environment.yaml](./environment.yaml): Environmental data and issues
