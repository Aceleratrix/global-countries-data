# Iran Data Directory

This directory contains 13 mandatory data files for Iran (Islamic Republic of Iran):
- `COUNTRY.md`: High-level country overview
- 11 structured YAML datasets covering economy, demographics, geography, politics, culture, history, military, energy, trade, infrastructure, and education
- This `README.md` index file

All data is sourced from verified public sources including World Bank, IMF, UN Statistics Division, and Iranian Statistical Center (2023-2024 estimates).
