# Qatar 🇶🇦 Data Directory

This directory contains all 13 mandatory data files for Qatar, structured as follows:
1. `COUNTRY.md` - Core country metadata
2. `README.md` - This overview file
3. 11 sector-specific YAML datasets covering economy, demographics, geography, government, culture, infrastructure, education, healthcare, energy, tourism, and military.

All data is sourced from official Qatari government reports, World Bank, IMF, and UN agencies (2023-2024 estimates).
