# Thailand Data Directory
This directory contains all 13 mandatory data files for Thailand as part of the global-countries-data repository.

## File List
1. `COUNTRY.md` - General overview and key facts
2. `README.md` - This file (directory metadata)
3. `general.yaml` - General country codes and identifiers
4. `geography.yaml` - Geographical data
5. `demographics.yaml` - Population and demographic statistics
6. `economy.yaml` - Economic indicators
7. `politics.yaml` - Political system and governance
8. `culture.yaml` - Cultural heritage and symbols
9. `tourism.yaml` - Tourism industry data
10. `infrastructure.yaml` - Infrastructure and transport
11. `education.yaml` - Education system metrics
12. `healthcare.yaml` - Healthcare system data
13. `environment.yaml` - Environmental statistics
