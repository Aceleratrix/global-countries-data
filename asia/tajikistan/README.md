# Tajikistan Data Repository
This folder contains 13 mandatory files with verified real data about Tajikistan, compliant with the global-countries-data repository structure.

## Files Included
1. COUNTRY.md - Comprehensive country profile with key metrics
2. README.md - This documentation file
3. basic_info.yaml - Core country identifiers and baseline stats
4. geography.yaml - Geographical and topographic data
5. demographics.yaml - Population structure and demographic indicators
6. economy.yaml - Economic performance and sector data
7. government.yaml - Political system and leadership information
8. culture.yaml - Cultural heritage, holidays, and traditions
9. history.yaml - Historical timeline and key events
10. tourism.yaml - Tourism resources and visitor data
11. education.yaml - Education system and enrollment metrics
12. healthcare.yaml - Healthcare infrastructure and health outcomes
13. infrastructure.yaml - Transportation, utilities, and connectivity data

All data is sourced from verified public sources including World Bank, CIA World Factbook, and Tajikistan State Statistics Committee (2020-2023 data).
