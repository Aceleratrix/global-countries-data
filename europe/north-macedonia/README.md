# North Macedonia Country Data
This directory contains structured, verified data for the Republic of North Macedonia 🇲🇰.

## File Structure
- `COUNTRY.md`: Overview of key country facts
- `README.md`: This file
- 11 YAML data files covering:
  1. `country.yaml`: Core country identifiers and metadata
  2. `geography.yaml`: Terrain, borders, natural features
  3. `demographics.yaml`: Population, ethnicity, language, religion
  4. `economy.yaml`: GDP, trade, currency, key sectors
  5. `government.yaml`: Political system, leaders, legislature
  6. `history.yaml`: Key historical events and timeline
  7. `culture.yaml`: Traditions, cuisine, arts, festivals
  8. `symbols.yaml`: Flag, coat of arms, anthem, national symbols
  9. `infrastructure.yaml`: Transport, energy, telecommunications
  10. `climate.yaml`: Weather patterns, average temperatures, precipitation
  11. `education.yaml`: Education system, literacy, universities

All data is sourced from verified public records (UN, World Bank, national statistics office).
