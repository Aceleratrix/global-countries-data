# San Marino Data Directory

This directory contains all 13 mandatory data files for San Marino as part of the global-countries-data repository.

## Files Included
- COUNTRY.md: Country overview and key facts
- README.md: This index file
- economy.yaml: Economic indicators and structure
- geography.yaml: Physical and political geography
- demographics.yaml: Population and social data
- government.yaml: Political system and governance
- culture.yaml: Cultural heritage and traditions
- infrastructure.yaml: Transport, utilities and digital infrastructure
- education.yaml: Education system and metrics
- healthcare.yaml: Healthcare system and outcomes
- defense.yaml: Military and security arrangements
- environment.yaml: Environmental data and protected areas
- tourism.yaml: Tourism industry and attractions

All data is sourced from verified public records including World Bank, UN, and official San Marino government sources.
