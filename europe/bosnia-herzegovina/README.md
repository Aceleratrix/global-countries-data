# Bosnia and Herzegovina Country Data

This directory contains all mandatory data files for Bosnia and Herzegovina as part of the global-countries-data repository.

## Files Included
- COUNTRY.md: Basic country overview
- README.md: This file
- basic_info.yaml: Core country identifiers and metadata
- geography.yaml: Geographic data including coordinates, borders, terrain
- population.yaml: Demographic statistics
- economy.yaml: Economic indicators and industry data
- languages.yaml: Official and spoken languages
- flags.yaml: National flag specifications and history
- borders.yaml: Border details with neighboring countries
- timezones.yaml: Timezone information
- maps.yaml: Map references and coordinate data
- translations.yaml: Country name translations
- other.yaml: Additional relevant country data
