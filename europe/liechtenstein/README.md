# Liechtenstein Country Data

This directory contains structured data for Liechtenstein (🇱🇮), the 178th largest economy globally with a nominal GDP of $0.05T.

## Files Included
- `COUNTRY.md`: Full country overview and key facts
- 11 structured YAML data files covering:
  1. `basic_info.yaml`: Core country identifiers and basic stats
  2. `geography.yaml`: Terrain, climate, borders, and physical features
  3. `demographics.yaml`: Population, ethnicity, religion, and health stats
  4. `economy.yaml`: GDP, industries, trade, and economic indicators
  5. `government.yaml`: Political structure, leadership, and governance
  6. `culture.yaml`: Languages, traditions, and cultural symbols
  7. `infrastructure.yaml`: Transport, energy, and telecommunications
  8. `education.yaml`: Schooling system and literacy rates
  9. `healthcare.yaml`: Medical infrastructure and public health stats
  10. `tourism.yaml`: Attractions, visitor stats, and travel info
  11. `symbols.yaml`: Flags, anthems, and national emblems

## Data Source
- Verified public domain and official government sources
- REST Countries API (where applicable)
- World Bank, IMF economic data
