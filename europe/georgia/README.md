# Georgia Country Data Directory
This directory contains 13 mandatory files with verified real-world data for Georgia (107th largest economy, $0.08T GDP).

## Files Included
1. `COUNTRY.md` - Main country overview and key facts
2. `README.md` - This file (directory overview)
3. `geography.yaml` - Geographic data (area, coordinates, borders, etc.)
4. `demographics.yaml` - Population, demographics, and related stats
5. `economy.yaml` - GDP, Gini index, and economic indicators
6. `government.yaml` - Sovereignty, UN membership, and governance info
7. `culture.yaml` - Demonyms, native names, and cultural identifiers
8. `language.yaml` - Official/primary languages and native language data
9. `currency.yaml` - Currency details (Lari, GEL, ₾)
10. `transport.yaml` - Driving side, car signs, and transport info
11. `maps.yaml` - Links to Google Maps and OpenStreetMap
12. `symbols.yaml` - Flag and coat of arms details
13. `timezones.yaml` - Timezone and weekly start day info

All data sourced from REST Countries API v3.1 and cross-verified with official sources.
