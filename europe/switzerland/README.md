# Switzerland 🇨🇭
This directory contains 13 mandatory data files for Switzerland (CH/CHE) as part of the global-countries-data repository.

## Files Included
1. `COUNTRY.md` - Basic country overview and key facts
2. `README.md` - This file
3. `demographics.yaml` - Population, age structure, migration data
4. `economy.yaml` - GDP, industries, employment
5. `geography.yaml` - Physical features, climate, regions
6. `government.yaml` - Political structure, federal system
7. `business.yaml` - Business environment, regulations
8. `tech.yaml` - Technology infrastructure, innovation
9. `telecom.yaml` - Telecommunications networks, internet
10. `legal.yaml` - Legal system, civil code
11. `fiscal.yaml` - Tax system, government revenue
12. `financial.yaml` - Banking, financial markets
13. `marketing.yaml` - Consumer market, media landscape

All data is sourced from official Swiss government statistics (BFS/OFS), World Bank, IMF, and UN databases (2024-2026 verified data).
