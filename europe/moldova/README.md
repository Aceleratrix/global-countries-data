# Moldova (Republic of Moldova) Data Repository
This directory contains structured, verified real-world data for Moldova, organized into 13 mandatory files:
- `COUNTRY.md`: High-level overview of key country facts
- 11 structured YAML datasets covering demographics, economy, geography, government, history, language, culture, infrastructure, education, healthcare, and environment
- This `README.md`: Directory summary

All data is sourced from verified public sources including the World Bank, UN Data, and the Government of Moldova.
