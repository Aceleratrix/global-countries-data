# Kosovo 🇽🇰
Data repository for Kosovo, 147th largest economy globally with $10.2B GDP.

## Files Structure
- COUNTRY.md: Main country overview
- basic_info.yaml: Core country identifiers
- geography.yaml: Geographic data
- population.yaml: Population details
- economy.yaml: Economic indicators
- languages.yaml: Language information
- flags.yaml: Flag assets and metadata
- borders.yaml: Neighboring countries
- timezones.yaml: Timezone data
- maps.yaml: Map links
- other.yaml: Additional country details

## Data Source
Real data fetched from REST Countries API (https://restcountries.com/v3.1/name/kosovo) plus verified secondary sources.
