# Sérvia - Dados Globais

Este diretório contém dados verificados da Sérvia, organizados em 11 arquivos YAML e documentação adicional.

## Arquivos
- `COUNTRY.md`: Informações básicas do país
- `business/`: Dados comerciais (import-export, real-estate, company-sizes)
- `marketing/`: Dados de marketing (digital-marketing)
- `legal/`: Leis (labor-law, internet-law)
- `tech/`: Tecnologia (ai-cloud-services)
- `demographics/`: Demografia (population)
- `telecom/`: Telecomunicações (phone-formats)
- `financial/`: Financeiro (banks)
- `fiscal/`: Fiscal (tax-codes)

Todos os dados são reais e verificados.
