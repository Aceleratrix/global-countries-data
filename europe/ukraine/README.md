# Ukraine Country Data

This folder contains 13 mandatory data files for Ukraine, including country overview, structured YAML datasets, and metadata.

## Files Included
1. COUNTRY.md - Country overview
2. README.md - This file
3. economy.yaml - Economic indicators
4. demographics.yaml - Population and demographic data
5. geography.yaml - Geographic information
6. politics.yaml - Political structure
7. culture.yaml - Cultural heritage
8. education.yaml - Education system
9. health.yaml - Healthcare data
10. infrastructure.yaml - Infrastructure details
11. military.yaml - Military information
12. environment.yaml - Environmental data
13. tourism.yaml - Tourism statistics
