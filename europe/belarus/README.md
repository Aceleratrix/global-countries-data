# Belarus Data Repository
This folder contains 11 structured YAML datasets and a country overview (COUNTRY.md) for Belarus, as part of the Global Countries Data project.

## Files
1. COUNTRY.md - High-level country overview
2. economy.yaml - Economic indicators and trade data
3. demographics.yaml - Population, age structure, and social stats
4. geography.yaml - Physical and political geography
5. politics.yaml - Government structure and leadership
6. culture.yaml - Traditions, language, and arts
7. education.yaml - Education system metrics
8. healthcare.yaml - Healthcare infrastructure and outcomes
9. transport.yaml - Transport networks (road, rail, air)
10. energy.yaml - Energy production and consumption
11. military.yaml - Military structure and personnel data
12. tourism.yaml - Tourism statistics and attractions
13. energy.yaml - Wait no, 11 YAMLs plus COUNTRY.md and this README is 13 total.
