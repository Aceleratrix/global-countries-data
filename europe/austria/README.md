# Austria - Country Data

This directory contains the complete set of 13 mandatory data files for Austria, following the standardized structure used for Spain, Russia, and Turkey.

## File List (13 total)
1. `COUNTRY.md` - Main country overview and key facts
2. `README.md` - This documentation file
3. `legal/internet-law.yaml` - Internet and digital legislation
4. `legal/labor-law.yaml` - Labor regulations and worker protections
5. `fiscal/tax-codes.yaml` - Tax rates, brackets, and codes
6. `financial/banks.yaml` - Banking sector and major institutions
7. `telecom/phone-formats.yaml` - Telecommunications numbering formats
8. `demographics/population.yaml` - Population statistics and trends
9. `tech/ai-cloud-services.yaml` - AI initiatives and cloud providers
10. `business/company-sizes.yaml` - Business size classifications
11. `business/real-estate.yaml` - Real estate market data
12. `business/import-export.yaml` - Trade statistics and partners
13. `marketing/digital-marketing.yaml` - Digital marketing landscape

All data reflects 2024 official statistics from Austrian government sources and Eurostat.
