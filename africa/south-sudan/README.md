# South Sudan Country Data

This directory contains structured, verified data files for the Republic of South Sudan, part of the global-countries-data repository.

## Files
- `COUNTRY.md`: Primary country overview with key facts
- 11 YAML data files covering detailed structured data:
  1. `country-profile.yaml` - Core country identifiers and basics
  2. `geography.yaml` - Physical geography, borders, terrain
  3. `demographics.yaml` - Population, ethnic groups, religion
  4. `economy.yaml` - GDP, industries, trade data
  5. `politics.yaml` - Government structure, political parties
  6. `history.yaml` - Key historical events timeline
  7. `culture.yaml` - Traditional music, cuisine, festivals
  8. `education.yaml` - Literacy, enrollment rates
  9. `healthcare.yaml` - Health metrics, challenges
  10. `infrastructure.yaml` - Roads, energy, telecoms
  11. `environment.yaml` - Protected areas, environmental issues
