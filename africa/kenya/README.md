# Kenya Data Directory
This directory contains 13 mandatory files for Kenya (96th largest economy, $0.10T GDP) as part of the global countries data repository.

## Files
1. COUNTRY.md: Main country overview with real API data  
2. README.md: This file  
3. 11 YAML files: Structured data on key country aspects  

## Data Source
All real data sourced from REST Countries API (https://restcountries.com/v3.1/name/kenya)
