# Kenya Data Directory 🇰🇪
This directory contains 13 comprehensive data files for Kenya (41st largest economy, $0.34T GDP):

## Files Included
1. COUNTRY.md - Main country overview
2. README.md - This file
3. economy.yaml - GDP, exports, currency data
4. demographics.yaml - Population, ethnic groups, languages
5. geography.yaml - Area, borders, climate
6. politics.yaml - Government, president, independence
7. culture.yaml - Heritage sites, cuisine, music
8. infrastructure.yaml - Airports, railways, internet access
9. education.yaml - Literacy, universities, enrollment
10. health.yaml - Life expectancy, healthcare spending
11. environment.yaml - National parks, endangered species
12. tourism.yaml - Tourist attractions, visitor numbers
13. sports.yaml - Popular sports, notable athletes

All data sourced from World Bank, IMF, UN, and Kenya National Bureau of Statistics (2023 estimates).
