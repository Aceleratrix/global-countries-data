# Djibouti Country Data

This directory contains structured, verified data for Djibouti (159th largest economy, $0.05T GDP) as part of the global-countries-data repository.

## Included Files
1. `COUNTRY.md` - Main country overview with key facts
2. 11 structured YAML datasets covering:
   - Geography
   - Demographics
   - Economy
   - Government
   - Culture
   - Infrastructure
   - Education
   - Healthcare
   - Environment
   - History
   - International Relations

All data is sourced from REST Countries API and verified public datasets.
