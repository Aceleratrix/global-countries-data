# Mauritania Data Directory
This directory contains all 13 mandatory data files for Mauritania (Islamic Republic of Mauritania), the 114th largest economy globally with $0.08T GDP.

## Files List
1. COUNTRY.md - Main country overview
2. README.md - This file
3. geography.yaml - Geographic data
4. demographics.yaml - Population/demographic data
5. economy.yaml - Economic indicators
6. government.yaml - Government structure
7. culture.yaml - Language/culture info
8. flags.yaml - Flag and coat of arms data
9. borders.yaml - Border country details
10. languages.yaml - Language specifics
11. currencies.yaml - Currency information
12. timezones.yaml - Timezone data
13. maps.yaml - Map links

All data sourced from REST Countries API and verified public records.
