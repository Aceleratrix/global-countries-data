# Egypt Data Repository
This directory contains 13 mandatory data files for Egypt, including country overview and 11 sector-specific YAML datasets.

## Files List
1. `COUNTRY.md` - High-level country overview
2. `README.md` - This file
3. `economy.yaml` - Economic indicators and key sectors
4. `demographics.yaml` - Population and social data
5. `geography.yaml` - Physical and political geography
6. `government.yaml` - Political structure and governance
7. `culture.yaml` - Cultural heritage and languages
8. `infrastructure.yaml` - Transport, energy, and telecom
9. `tourism.yaml` - Tourism sector metrics
10. `energy.yaml` - Energy production and consumption
11. `education.yaml` - Education system data
12. `healthcare.yaml` - Healthcare infrastructure
13. `military.yaml` - Military capabilities

All data sourced from World Bank, IMF, and Egyptian government reports (2023-2024 est.).
