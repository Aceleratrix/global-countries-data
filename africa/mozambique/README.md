# Mozambique Data Repository

This directory contains 13 mandatory structured data files for Mozambique (🇲🇿), the 112th largest economy globally with $0.08T GDP.

## Files Included
1. COUNTRY.md - Core country overview
2. basics.yaml - Fundamental country identifiers
3. geography.yaml - Geographic and border data
4. economy.yaml - Economic indicators and GDP data
5. demographics.yaml - Population and demographic stats
6. government.yaml - Political and governance info
7. culture.yaml - Language, demonyms, and cultural data
8. infrastructure.yaml - Transport and infrastructure details
9. education.yaml - Education system data
10. health.yaml - Health sector indicators
11. environment.yaml - Environmental and climate data
12. international.yaml - UN membership, borders, and intl. relations
13. README.md - This file

## Data Sources
- REST Countries API v3.1
- World Bank
- United Nations
