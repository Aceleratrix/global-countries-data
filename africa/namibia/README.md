# Namibia Country Data

This directory contains complete structured data for Namibia (Republic of Namibia), 109th largest economy globally with $0.08T GDP.

## Files Included
1. `COUNTRY.md` - Main country overview with all key information
2. `README.md` - This file, directory documentation
3. `general.yml` - General country metadata
4. `geography.yml` - Geographic data
5. `population.yml` - Population statistics
6. `economy.yml` - Economic indicators
7. `languages.yml` - Official and native languages
8. `currencies.yml` - Currency information
9. `borders.yml` - Border countries data
10. `timezones.yml` - Timezone information
11. `flags.yml` - Flag assets and descriptions
12. `coat_of_arms.yml` - Coat of arms assets
13. `translations.yml` - Country name translations

## Data Source
All data sourced from REST Countries API v3.1 (https://restcountries.com) and verified against official Namibian government sources.
