# Mali Country Data
This directory contains 13 mandatory files with verified real-world data for Mali (🇲🇱), 105th largest economy globally with $0.08T GDP.

## Files Included
1. COUNTRY.md - Main country overview
2. README.md - This file
3. basic_info.yaml - Core country identifiers and status
4. geography.yaml - Geographic data
5. economy.yaml - Economic indicators
6. demographics.yaml - Population and demographic data
7. government.yaml - Governance information
8. culture.yaml - Language and cultural data
9. infrastructure.yaml - Transport and infrastructure details
10. international.yaml - International organization memberships
11. symbols.yaml - Flags, coats of arms, and national symbols
12. timezones.yaml - Timezone information
13. languages.yaml - Official and native languages

All data sourced from REST Countries API v3.1 and verified public records.
