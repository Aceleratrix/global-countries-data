# Tunisia Data Folder
This folder contains 13 mandatory files with comprehensive, verified real-world data about Tunisia:
- `COUNTRY.md`: High-level country overview
- 11 structured YAML data files covering economy, demographics, geography, government, culture, infrastructure, education, health, energy, tourism, and sports
- This README.md

All data is sourced from IMF, World Bank, UN, and Tunisian National Institute of Statistics (INS) 2023-2024 reports.
