# Cabo Verde (Cape Verde) Data Repository
This directory contains structured data for Cabo Verde (Cape Verde), the 156th largest economy globally with $0.05T GDP.

## Files Included
1. COUNTRY.md - Core country overview
2. README.md - This file
3. geography.yaml - Geographic and territorial data
4. demographics.yaml - Population and demographic metrics
5. economy.yaml - Economic indicators and GDP data
6. government.yaml - Political structure and governance
7. culture.yaml - Language, religion, and cultural traits
8. infrastructure.yaml - Transport, energy, and telecoms
9. education.yaml - Education system and statistics
10. health.yaml - Healthcare system and health metrics
11. environment.yaml - Climate, biodiversity, and environmental policies
12. tourism.yaml - Tourism sector data
13. trade.yaml - Trade partners and export/import data

## Data Source
All data is sourced from verified public datasets including REST Countries API, World Bank, and national statistics offices.
