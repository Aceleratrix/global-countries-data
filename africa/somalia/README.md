# Somalia Data Directory
This directory contains all 13 mandatory data files for Somalia (142nd largest economy, $0.06T GDP).

## Files Included
1. COUNTRY.md - Main country overview
2. README.md - This file
3. demographics.yaml - Population, age structure, ethnicity data
4. economy.yaml - GDP, currency, trade data
5. geography.yaml - Area, borders, coastline, terrain
6. government.yaml - Political structure, independence, leaders
7. infrastructure.yaml - Transport, energy, telecom data
8. language.yaml - Official languages, dialects
9. culture.yaml - Religion, traditions, arts
10. climate.yaml - Weather patterns, precipitation
11. history.yaml - Key historical events
12. sports.yaml - Popular sports, achievements
13. symbols.yaml - Flag, anthem, national emblems

## Data Source
REST Countries API + verified public domain references
