# Niger Data Directory
This directory contains 13 mandatory files for Niger (🇳🇪), 115th largest economy globally with $0.08T GDP.

## Files Included
1. COUNTRY.md - Country overview and key facts
2. README.md - This file
3. basics.yaml - Core country identifiers and metadata
4. geography.yaml - Geographic data (area, borders, coordinates)
5. demographics.yaml - Population, demographics, and social data
6. economy.yaml - GDP, currency, economic indicators
7. government.yaml - Political structure and governance data
8. culture.yaml - Language, religion, cultural facts
9. infrastructure.yaml - Transport, energy, and infrastructure data
10. health.yaml - Health indicators and systems
11. education.yaml - Education metrics and systems
12. security.yaml - Military, safety, and security data
13. environment.yaml - Environmental facts and protected areas

All data sourced from REST Countries API and verified public datasets.
