# Eritrea Country Data

This directory contains 13 mandatory files with comprehensive, real-world data on Eritrea, including geographic, economic, demographic, and cultural information.

## Files Included
1. `COUNTRY.md` - Main country overview
2. `README.md` - This file
3. `geography.yaml` - Geographic and location data
4. `economy.yaml` - Economic indicators and currency info
5. `demographics.yaml` - Population and ethnic/language data
6. `government.yaml` - Government structure and independence details
7. `culture.yaml` - Cultural, religious, and holiday information
8. `infrastructure.yaml` - Transportation and infrastructure details
9. `education.yaml` - Education system and literacy rates
10. `healthcare.yaml` - Healthcare statistics and life expectancy
11. `environment.yaml` - Terrain, climate, and natural resources
12. `military.yaml` - Military structure and expenditure
13. `tourism.yaml` - Tourism attractions and visitor information
