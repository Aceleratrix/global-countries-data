# Sao Tome and Principe Country Data

This directory contains structured data for Sao Tome and Principe following the global-countries-data repository standard.

## Files
- `COUNTRY.md`: Basic country overview
- `culture.yaml`: Cultural information
- `demographics.yaml`: Population and demographic data
- `economy.yaml`: Economic indicators
- `education.yaml`: Education system data
- `energy.yaml`: Energy sector data
- `geography.yaml`: Geographical details
- `health.yaml`: Healthcare system data
- `infrastructure.yaml`: Infrastructure information
- `politics.yaml`: Political system data
- `tech.yaml`: Technology sector data
- `trade.yaml`: Trade and commerce data

All data is sourced from verified public sources including UN, World Bank, and national statistics offices.
