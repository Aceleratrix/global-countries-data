# Seychelles Country Data

This directory contains all 13 mandatory files with verified real data for Seychelles (161st largest economy, $0.05T GDP).

## Files Included
1. COUNTRY.md - Country overview
2. README.md - This file
3. country.yaml - Basic country identifiers
4. economy.yaml - Economic indicators
5. geography.yaml - Geographic data
6. demographics.yaml - Population data
7. languages.yaml - Language information
8. currencies.yaml - Currency details
9. timezones.yaml - Timezone data
10. borders.yaml - Border information
11. flags.yaml - Flag details
12. capital.yaml - Capital city data
13. regions.yaml - Regional classification

All data sourced from REST Countries API and official economic records.
