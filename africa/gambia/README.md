# Gambia (Republic of the Gambia) Data
This folder contains 13 mandatory structured data files for Gambia, including country metadata, economic stats, geography, demographics, and official symbols.

## Files Included
1. COUNTRY.md - Core country overview
2. README.md - This file
3. 11 structured YAML data files (basic_info.yaml, geography.yaml, demographics.yaml, economy.yaml, government.yaml, languages.yaml, currencies.yaml, flags.yaml, borders.yaml, timezones.yaml, symbols.yaml)

## Key Stats
- Global GDP Rank: 145th ($0.05T)
- Population: ~2.42 million
- Capital: Banjul
- Currency: Dalasi (GMD)
