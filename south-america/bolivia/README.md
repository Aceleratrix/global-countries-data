# Bolivia Country Data
This directory contains structured, verified data about Bolivia, organized into 11 YAML datasets and two Markdown files (COUNTRY.md and this README).

## Files
- `COUNTRY.md`: High-level country overview
- `basics.yaml`: Core identifiers and basic stats
- `geography.yaml`: Physical geography and natural features
- `demographics.yaml`: Population, ethnicity, and social indicators
- `economy.yaml`: Economic metrics and industry data
- `politics.yaml`: Government structure and political information
- `culture.yaml`: Cultural symbols, traditions, and UNESCO sites
- `history.yaml`: Key historical events timeline
- `climate.yaml`: Climate zones and weather patterns
- `infrastructure.yaml`: Transport, energy, and telecom infrastructure
- `education.yaml`: Education system and literacy statistics
- `health.yaml`: Public health and healthcare indicators

## Data Sources
All data sourced from verified public datasets: World Bank, UN Statistics Division, Bolivian National Institute of Statistics (INE), and UNESCO.
