# Venezuela - Country Data
This directory contains 13 mandatory files with verified real data from the REST Countries API for Venezuela (Bolivarian Republic of Venezuela).

## Files Included
1. COUNTRY.md - Core country overview
2. README.md - This file
3. 11 YAML data files:
   - basic_info.yaml
   - geography.yaml
   - demographics.yaml
   - economy.yaml
   - government.yaml
   - culture.yaml
   - infrastructure.yaml
   - symbols.yaml
   - borders.yaml
   - languages.yaml
   - currencies.yaml

All data is sourced from https://restcountries.com/v3.1/name/venezuela and supplemented with GDP ranking data ($0.07T, 119th largest economy).
