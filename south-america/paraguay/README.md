# Paraguay Country Data
This directory contains verified, real-world data about Paraguay structured into 13 mandatory files:
1. `COUNTRY.md` (overview of key country facts)
2. `README.md` (this file)
3. 11 structured YAML datasets covering:
   - Geography
   - Demographics
   - Economy
   - Government
   - Culture
   - History
   - Education
   - Healthcare
   - Infrastructure
   - Environment
   - Tourism

All data is sourced from verified public records (World Bank, UN, Paraguayan government sources) as of 2024.
