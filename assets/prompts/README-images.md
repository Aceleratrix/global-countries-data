## Usage Instructions

These prompts are designed for:
- **Midjourney**: Add `--ar 3:1` for hero banner, `--ar 4:3` for diagrams
- **DALL-E 3**: Use as-is for high-quality technical illustrations
- **Stable Diffusion**: May need simplification for complex multi-element prompts
- **ComfyUI**: Can handle complex node-based generation

Save generated images to: `assets/images/` directory.

## Image Naming Convention

- `hero-banner.png` - Main README header
- `data-structure.png` - Schema/architecture diagram  
- `multi-agent-integration.png` - Integration diagram
- `coverage-progress.png` - World map with progress
- `country-org-chart.png` - Folder structure visualization

## Notes

- All images should use **dark theme** (#0f172a, #1e293b) to match modern GitHub README aesthetics
- Use **neon accents** (cyan #06b6d4, magenta #d946ef, green #10b981) for tech feel
- Keep text minimal in images (use image captions in README instead)
- High contrast for readability on both light/dark GitHub themes
